dftgwbse
********

the `dftgwbse` calculator optionally takes an *xml* file that have the following hierarchical
structure:

::

   dftgwbse
   ├── general dftgwbse options
   ├── DFTpackage
   │   └── general dftpackage options
   ├── gwbse_engine
   │   ├── task
   │   └── gwbse options
   ├── geometry optimization
   │   └── geometry optimization options
